# PHACS-NMR

Browser application for PHA polymer characterization from Bruker TopSpin NMR exports.
Identifies hydroxyacid monomers, estimates molar composition, and resolves chain microstructure.

**Audience:** Polymer scientists with an NMR background
**Version:** 3 (June 2026)

## Usage

Open `PHACS-NMR.html` in any modern browser (Chrome, Edge, Firefox, Safari). No installation required. All computation runs locally — your data never leaves the machine.

An internet connection is only needed on first load to cache charting and PDF libraries from CDN.
